# SIMPEL-Fella

SIMPEL-Fella adalah prototype frontend sistem informasi manajemen penjadwalan dan presensi Fella Swimming berbasis web.

## Fokus Sistem

Sistem ini berfokus pada:
- Pengelolaan jadwal latihan
- Presensi siswa
- Pemantauan sisa sesi latihan
- Data siswa
- Data pelatih
- Laporan operasional sederhana

## Role Pengguna

1. Admin
2. Pelatih
3. Siswa/Orang Tua

## Teknologi

- HTML
- CSS
- JavaScript

## Status Project

Prototype frontend untuk kebutuhan tugas Pengembangan Aplikasi Berbasis Web.
